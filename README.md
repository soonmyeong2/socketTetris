The Tetris source code is here https://github.com/troglobit/tetris.

This code is an online collaboratin-tetris based on Linux socket communication.


## 설명

* 리눅스 기반으로 한 소켓 프로그래밍, IPC, syncronazation, File I/O 기법을. 적절히 사용하여 Tetris게임을 제작한다.
* Tetris게임을 여러 클라이언트들이 같은 map으로 collaboration형식으로 게임을 진행하여 점수를 쌓고, 점수를 얻으면 랭킹이 몇 위인지 알 수 있게 한다.
